# 📡 Telegram Online Tracker Bot

Бот для отслеживания онлайн-статуса пользователей Telegram. Автоматически уведомляет владельца, когда отслеживаемые пользователи заходят в сеть или выходят из неё.

## ✨ Возможности

- 🔔 Мгновенные уведомления о входе и выходе из сети
- 👥 Отслеживание нескольких пользователей одновременно
- ➕ Добавление и удаление пользователей командами прямо в боте
- ⏸️ Пауза и возобновление мониторинга без перезапуска
- 📊 Просмотр текущего статуса всех отслеживаемых
- 🔒 Доступ только для владельца (по Chat ID)

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/DenMobile206/telegram-tracker.git
cd telegram-tracker
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Получить API_ID и API_HASH

1. Зайдите на [my.telegram.org](https://my.telegram.org)
2. Войдите в свой аккаунт Telegram
3. Перейдите в раздел **API development tools**
4. Создайте приложение и скопируйте `api_id` и `api_hash`

### 4. Создать бота через @BotFather

1. Напишите боту [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot` и следуйте инструкциям
3. Скопируйте полученный **токен** бота

### 5. Узнать свой Chat ID

1. Напишите боту [@userinfobot](https://t.me/userinfobot)
2. Скопируйте ваш **Chat ID**

### 6. Заполнить настройки в tracker_bot.py

Откройте файл `tracker_bot.py` и заполните значения в начале файла:

```python
API_ID = 12345678              # Ваш api_id с my.telegram.org
API_HASH = 'your_api_hash'    # Ваш api_hash с my.telegram.org
BOT_TOKEN = '7123456789:AAH...'  # Токен от @BotFather
MY_CHAT_ID = 123456789         # Ваш Chat ID от @userinfobot
TRACK_USERS = ['username1', 'username2']  # Юзернеймы без @
```

### 7. Запустить скрипт

```bash
python tracker_bot.py
```

При первом запуске Telethon попросит ввести номер телефона и код подтверждения для авторизации в Telegram.

## 📋 Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список всех команд |
| `/status` | Текущий онлайн-статус всех отслеживаемых |
| `/list` | Список отслеживаемых пользователей |
| `/add username` | Добавить пользователя в список |
| `/remove username` | Убрать пользователя из списка |
| `/pause` | Приостановить мониторинг |
| `/resume` | Возобновить мониторинг |

## ⚠️ Важные предупреждения

- 🔑 **Не делитесь файлом `*.session`** — он содержит авторизационные данные вашего аккаунта Telegram
- 💻 **Скрипт работает только пока компьютер включён** — для постоянной работы используйте VPS или сервер
- 👁️ **Скрытый статус не отслеживается** — если пользователь скрыл статус «был в сети», бот не сможет получить информацию о его активности
