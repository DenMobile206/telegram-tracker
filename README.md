# 📡 Telegram Online Tracker

Telegram-бот, который отслеживает онлайн-статус указанных пользователей и отправляет уведомления владельцу, когда они заходят в сеть или выходят из неё.

## ✨ Что умеет бот

- 🟢 Уведомляет, когда отслеживаемый пользователь заходит в сеть
- 🔴 Уведомляет, когда пользователь выходит из сети
- 📊 Показывает текущий статус всех отслеживаемых пользователей (`/status`)
- 👥 Управление списком отслеживаемых прямо через бота (`/add`, `/remove`)
- ⏸ Возможность приостановить и возобновить мониторинг (`/pause`, `/resume`)
- 🔒 Все команды доступны только владельцу бота

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/DenMobile206/telegram-tracker.git
cd telegram-tracker
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Получить API_ID и API_HASH

1. Откройте [https://my.telegram.org](https://my.telegram.org) и войдите в аккаунт
2. Перейдите в раздел **API development tools**
3. Создайте приложение и скопируйте **App api_id** и **App api_hash**

### 4. Создать бота через @BotFather

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный **токен** (выглядит как `123456789:ABCdef...`)

### 5. Узнать свой Chat ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте любое сообщение
3. Скопируйте число из поля **Id**

### 6. Заполнить настройки в `tracker_bot.py`

Откройте файл `tracker_bot.py` и заполните блок настроек в верхней части:

```python
API_ID    = 12345678        # Ваш api_id с my.telegram.org
API_HASH  = "abcdef1234..."  # Ваш api_hash с my.telegram.org
BOT_TOKEN = "123456:ABC..."  # Токен от @BotFather
MY_CHAT_ID = 987654321      # Ваш Chat ID от @userinfobot
TRACK_USERS = ["username1", "username2"]  # Юзернеймы без @
```

### 7. Запустить

```bash
python tracker_bot.py
```

При первом запуске Telethon попросит ввести номер телефона и код подтверждения для авторизации аккаунта. После этого создастся файл `tracker.session` — **не передавайте его никому!**

После успешного запуска бот пришлёт сообщение:
> 🚀 Трекер запущен! Отслеживаю: N чел.

## 📋 Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/status` | Текущий статус всех отслеживаемых пользователей |
| `/list` | Список отслеживаемых пользователей |
| `/add username` | Добавить пользователя в отслеживание |
| `/remove username` | Убрать пользователя из отслеживания |
| `/pause` | Приостановить мониторинг |
| `/resume` | Возобновить мониторинг |

---

## ⚠️ Важные предупреждения

- **Не передавайте файл `tracker.session` никому** — он содержит данные авторизации вашего Telegram-аккаунта
- **Скрипт работает только пока запущен** — если выключить ПК или закрыть терминал, мониторинг прекратится
- **Скрытый статус не отслеживается** — если пользователь скрыл статус «был в сети» в настройках приватности, отследить его не получится
- Скрипт использует ваш личный Telegram-аккаунт через Telethon; злоупотребление может привести к ограничениям со стороны Telegram
